# Install Termux
```
• pkg update && pkg upgrade
• pkg install nodejs ffmpeg git libwebp wget
• termux-setup-storage
• git clone https://github.com/Z7QX/Seven-MD.git
• cd Seven-MD
• npm i
• yarn install 
• npm start

Pairing? use $ node index.js --pairing
```
